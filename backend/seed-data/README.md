# Seed data for reviewers

This folder is generated by **export** and used by **seed**.

- **Generate (author, once):** From `backend/` with your DB that has the products, run:
  ```bash
  npm run export-seed
  ```
  This creates `export.json` (users, categories, products, ratings) and copies product images into `seed-data/uploads/`.
  **If you use Docker:** the image files live in the container volume, not in host `backend/uploads/`. So copy them to the host first, then run export-seed:
  ```bash
  # From project root (e.g. i-love-shopping1)
  docker compose cp backend:/app/backend/uploads/. backend/uploads/
  cd backend && npm run export-seed
  ```
  
- **Import (reviewers):** After `docker compose up -d` (or local migrations), run:
  ```bash
  docker compose exec backend npx prisma db seed
  ```
  or locally: `cd backend && npx prisma db seed`. All seeded users get password **Demo123!**.
